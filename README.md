# OATF Admin Portal Mockup

Static clickable concept for a private Out at the Fair® + Hyperion LA operations portal.

Open `index.html` locally or publish this folder through GitHub Pages.

This mockup includes:
- Demo login
- Season dashboard
- Fair workspaces
- Task board
- Talent pipeline
- Calendar
- Files
- Activity log
- Day-of command mode

No accounts, databases, uploads, email notifications, or publishing functions are connected.
